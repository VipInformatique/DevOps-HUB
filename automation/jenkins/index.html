Jenkins Pipeline: Automatic Nginx Deployment and Testing

This Jenkins pipeline performs an automated deployment and test of an Nginx container on a RHEL system. The pipeline has the following steps:

  • ⚙️ System Update: Update the system using dnf update -y.
  • 📥 Pull Nginx Image: Download the latest Nginx image from Docker Hub.
  • 🚀 Run and Test Nginx:
    • Start Nginx container mapped to port 8088.
    • Check if Nginx responds with the welcome page.
    • Clean up: Stop and remove the container.
